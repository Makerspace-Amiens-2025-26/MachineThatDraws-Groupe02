# Introduction

Bienvenue dans la documentation du projet Machine that draws. Ce site a pour but de fournir toutes les informations nécessaires pour comprendre, utiliser et reproduire efficacement notre projet.

## Contexte du Projet

Ce projet a été réalisé dans le cadre d’un travail scolaire visant à concevoir et réaliser une machine à écrire automatisée. 

La machine repose sur un système de déplacement sur deux axes (X et Y) permettant de positionner un outil d’écriture au-dessus d’une feuille afin de tracer des caractères ou des formes. 

L’objectif était de concevoir un système fonctionnel mêlant mécanique, électronique et programmation, tout en respectant les contraintes imposées par le projet. 

## Objectifs du Projet

Les objectifs du projet étaient : 

-Concevoir une machine capable de dessiner sur une feuille de papier 

-Mettre en œuvre un système de déplacement précis sur les axes X et Y 

-Utiliser des moteurs pas à pas pour assurer la précision des mouvements 

-Programmer le pilotage de la machine 

-Documenter le projet via un site web et un dépôt GitHub 

# Existant

Avant la conception, une étude de solutions existantes a été réalisée. 

Les machines de type traceur, imprimante 3D ou CNC utilisent généralement des moteurs pas à pas, des courroies et des rails pour assurer des déplacements précis. 

Cette étude nous a permis de nous inspirer de ces systèmes afin d’adapter une solution simplifiée et accessible répondant aux objectifs du projet. 

# Cahier des Charges

La machine devait répondre aux contraintes suivantes : 

-Déplacement précis sur deux axes (X et Y) 

-Capacité à dessiner sur une feuille de format standard 

-Structure stable et rigide 

-Utilisation de moteurs pas à pas 

-Pilotage via une carte électronique 

-Facilité d’assemblage et de maintenance 

-Coût limité et composants accessibles 

-Sécurité de l’utilisateur lors de l’utilisation
